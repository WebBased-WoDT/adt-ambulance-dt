DTKnowledgeGraph
Link copied to clipboard
data class DTKnowledgeGraph(val dtUri: String, val predicates: List<Pair<Property, Node>> = listOf())

It models the Digital Twin Knowledge graph of a Digital Twin, identified by its dtUri, in an abstract way. Its objective is to serve as the base to represent semantic data, in particular its predicates, about a Digital Twin and not to be a generic implementation of a Knowledge Graph. Its main usage will be as a DTO to be converted in different KG models of different libraries.
